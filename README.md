# steuerspirale
Interaktive Steuerspirale (D3.js) mit Live-Daten aus der GENESIS-Online API des Statistischen Bundesamts (Tabelle 71211-0001). Archimedische Spirale, Hover-Tooltips, konfigurierbarer Spiralstart. Python-Backend für automatischen Datenabruf. GitHub Actions Cron + manueller Trigger für OpenClaw-Bot.
